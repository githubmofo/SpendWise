# 💰 SpendWise

### Smart Expense & Spending Behavior Analyzer using Machine Learning

---

## 📌 Overview

SpendWise is a data-driven financial analytics project that analyzes user spending behavior, predicts expenses, and provides meaningful insights using Machine Learning techniques.

The project uses real-world data collected through Google Forms and applies a complete data science pipeline including data cleaning, feature engineering, visualization, and predictive modeling.

---

## 🎯 Objectives

* Analyze daily expense patterns
* Predict expense amount using ML models
* Identify overspending behavior
* Segment users based on spending habits

---

## 📊 Dataset

* Source: Google Forms (custom dataset)
* Records: 177 entries → cleaned to 167
* Features include:

  * Age, Income, Expense Amount
  * Expense Category, Payment Method
  * Budget behavior, Saving goals

---

## 🧠 Project Pipeline

1. Data Collection (Google Forms)
2. Data Cleaning & Preprocessing
3. Feature Engineering
4. Exploratory Data Analysis (EDA)
5. Model Training (5 Algorithms)
6. Model Evaluation & Selection
7. Prediction & Insights
8. User Segmentation (KMeans)

---

## 🤖 Machine Learning Models Used

* Linear Regression

* K-Nearest Neighbors (KNN)
